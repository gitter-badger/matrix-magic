Program: Able to multiply, add, and subtract matrices

Author: Linda Mummy, September 2014

matrix.html/matrixCSS.css/matrixPage.js - provides GUI for matrix calculator (GUI a work in progress)

matrixLibrary - reusable library of matrix functions including the Matrix object, and functions printMatrix(A), multiplyMatrices(A,B), addMatrices(A,B), subtractMatrices(A,B), and constantMultiply(A,c) which accept Matrices A/B and constant c as parameters.
